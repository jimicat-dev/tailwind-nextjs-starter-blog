# 我的第一篇文章

## Guo-En Blog 誕生

今天是一個難得很閒的上班日，不用開 Daily Meeting 的早晨還真不習慣。

人真的只要一閒下來就會胡思亂想，於是我就逛起 namecheap 順手搜了自己的名字。

「咦？只要 400 塊台幣/年喔？買下去了吧」，嗯，就順手跟 Claude 一同協作完成初版了。

以下是今天做的事情：

1. [namecheap](https://www.namecheap.com/) 買 `guoen.dev` domain
2. 選擇網站 Template repo -> clone 下來改/做 logo -> git push
3. Vercel 部署

中間遇到 Template 長期沒有維護導致版本錯誤，怎麼修都修不好的狀況，最後只好換成現在的 Template。

## 未來網站方向

1. 技術文章：工作時用到的技術及心得
2. Side project：記錄實作過程、遇到的新技術
3. 生活：不希望生活只剩下 coding，偶而記錄樸實無華的人生
